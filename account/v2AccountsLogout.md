# V2登出

#### 功能说明

修改用户在设备上的状态为登出状态

guid，请求cookie中需要传递设备的guid

aat，请求cookie中需要传递用户的accessToken

#### 调用接口说明

* #### HTTP配置

| 配置项   | 取值                           |
| -------- | ------------------------------ |
| URL      | \[域名\]/v2/accounts/logout    |
| 请求头部 | application/json;charset=utf-8 |
| HTTP方式 | POST                           |

* #### 输入参数说明

无

* #### HTTP 请求示例

```

```

* #### 返回参数

无

* #### 响应示例：

```json
{
    "code": "1",
    "note": "操作成功"
}
```

* #### 异常返回值

| 错误码 | 提示信息       | 备注                                       |
| ------ | -------------- | ------------------------------------------ |
| 3      | 必填参数为空。 |     accessToken为空， accessToken不存在                 |
| 160    | 设备未注册     | cookie中的guid为空，guid不在服务器中不存在 |
