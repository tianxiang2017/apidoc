## 下载固件包文件

#### 功能说明

下载一个固件包。
请求路径中的firmwareId是固件包id

#### 调用接口说明

* #### HTTP配置

| 配置项 | 取值 |
| --- | --- |
| URL | \[域名\]/firmwares/download/{firmwareId}|
| 请求头部 | Content-Type:application/json;charset=utf-8 |
| HTTP方式 | GET |

* #### 输入参数说明

无


* #### HTTP 请求示例


```
http://localhost:9001/firmwares/download/143
```



* #### 返回参数
固件包文件




* #### 响应示例：

固件包文件

* #### 异常返回值

无
