## 删除角色

#### 功能说明
删除指定角色ID的角色。同时删除这个角色拥有的权限，权限本身不删除。拥有此角色的用户将失去这个角色。


#### 调用接口说明

* #### HTTP配置

| 配置项 | 取值 |
| --- | --- |
| URL | \[域名\]/role/delete|
| 请求头部 | Content-Type:application/json;charset=utf-8 |
| HTTP方式 | POST|

* #### 输入参数说明

| 参数名称 | 参数类型 | 是否必须 | 参数描述 |
| :--- | :--- | :--- | :--- |
|id|Integer\(10\) | 否 | 角色ID|



* #### HTTP 请求示例

```json
{
	"roleId": 17
}
```


* #### 返回参数
无


* #### 响应示例：

```json
{
    "code": "1000",
    "note": "Operate success",
}
```
* #### 异常返回码
无
