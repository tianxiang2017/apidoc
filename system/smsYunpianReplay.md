# 云片短信回复内容

#### 功能说明

用户回复短信后，回调此接口

#### 调用接口说明

* #### HTTP配置

| 配置项 | 取值 |
| --- | --- |
| URL | \[域名\]/sms/yunpian/reply |
| 请求头部 |  |
| HTTP方式 | POST |

* #### 输入参数说明

| 参数名称 | 参数类型 | 是否必须 | 描述 |
| :--- | :--- | :--- | :--- |
| sms_status| String | 否 | 短信状态字符串。字符串内容是一个数组，一个手机号对应一个元素|
